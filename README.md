
Hangman Game 🎮

A simple and interactive Hangman Game built using Python. The player has to guess the hidden word letter by letter before all chances are exhausted.

📌 Features Random word selection User-friendly command line interface Tracks guessed letters Limited attempts system Beginner-friendly Python project Easy to customize word list 🛠️ Technologies Used Python 3 📂 Project Structure hangman-game/ │── hangman.py │── README.md ▶️ How to Run the Project Install Python on your system. Clone this repository or download the files. Open terminal or command prompt. Run the following command: python hangman.py 🎯 How to Play The game will randomly choose a word. You need to guess one letter at a time. Correct guesses reveal the letter position. Wrong guesses reduce your remaining attempts. The game ends when: You guess the complete word 🎉 OR your attempts become zero ❌ 💡 Example Word: _ _ _ _ _ Enter a letter: a
