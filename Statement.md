# Problem Statement: Hangman Game using Python

## Introduction
The Hangman Game is a classic word-guessing game developed using Python programming language. The objective of this project is to create an interactive console-based game where the user tries to guess a hidden word by entering one letter at a time within a limited number of attempts.

## Problem Statement
Design and develop a Python-based Hangman Game that allows users to guess randomly selected words from a predefined list. The program should provide hints through blank spaces representing letters, validate user input, track incorrect guesses, and determine whether the player wins or loses based on the number of attempts.

## Objectives
- To implement a fun and interactive word guessing game.
- To practice Python programming concepts such as loops, conditional statements, functions, lists, and string manipulation.
- To improve logical thinking and problem-solving skills.
- To create a user-friendly command-line game experience.

## Functional Requirements
- Randomly select a word from a predefined word list.
- Display blank spaces for unguessed letters.
- Accept single-letter input from the user.
- Validate user input to ensure correctness.
- Track correct and incorrect guesses.
- Limit the number of wrong attempts.
- Display win/loss messages.

## Technologies Used
- Python
- Random Module
- Loops
- Conditional Statements
- Lists and Strings

## Expected Outcome
A fully functional Python Hangman Game that provides an engaging experience while demonstrating core Python programming concepts.
